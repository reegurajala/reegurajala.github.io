01

Tech Layoffs Analysis (2020–2026)

End-to-end SQL and Python analysis of 4,361 global tech layoff events sourced from layoffs.fyi, covering the full arc of the mass layoff wave from COVID-19 through 2026. Wrote 7 analytical SQL queries using CTEs, nested window functions, and time-series aggregations to identify industry, company, and geographic patterns.

Key findings

2023 peaked at 264,320 layoffs across 749 companies. Amazon led all companies with 58,124 cuts across 13 separate rounds. 2026 is already on pace to exceed 2024 with 92,272 layoffs in just 4 months.

02

Hospital Readmissions Analysis

End-to-end SQL, Python, and Power BI analysis of 18,330 CMS hospital records using window functions, CTEs, and CASE WHEN tier classification to identify excess readmission patterns across U.S. facilities. Built an interactive 3-page Power BI dashboard with state-level comparisons, condition breakdowns, and hospital-level filtering.

Key findings

Hip/Knee replacements are the highest-risk condition at 23% of hospitals classified High Risk. Massachusetts flagged as a systemic underperformer across every analysis — state averages, rate gaps, and worst hospital rankings.

03

Pokemon Competitive Analysis

A multi-part analysis across 1,025 Pokemon examining competitive viability through base stats, Smogon tier placement, and move pool access. Merged data from PokeAPI and a custom Smogon web scraper, then built a custom Power Score metric to test what actually drives tier placement.

Key findings

Move pool access is no longer a competitive differentiator after 30 years of TM distribution. Stats and abilities are the real drivers. Gen 9 Paradox mechanics produce regular Pokemon that rival Legendaries.
